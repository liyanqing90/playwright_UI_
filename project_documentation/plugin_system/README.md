详细步骤请参考 [插件开发指南](./development_guide.md)。

### 插件管理

```python
from src.core.plugin_manager import PluginManager

# 创建插件管理器
plugin_manager = PluginManager()

# 加载插件
plugin_manager.load_plugin('path/to/plugin')

# 列出已加载的插件
loaded_plugins = plugin_manager.list_plugins()
```
